# 外部 Agent 任务：假说 A × B 交叉推演

> 项目：LLM 与人类直觉：进化路线探索 | 阶段：第三阶段 前瞻推演
> 
> 本任务要求对**假说 A（LLM 作为直觉假肢）**和**假说 B（LLM 削弱人类直觉）**进行交叉推演。
> 
> **主项目文档**（背景参考）：[LLM_Intuition_Exploration.md](LLM_Intuition_Exploration.md) v0.9
> 
> **交叉推演的含义**：A 和 B 不是独立的两个趋势，它们相互构成一个正反馈循环。(1) A（LLM 补人的直觉盲区）→ 人更依赖 LLM → B（人类直觉退化）→ 人直觉更差 → 更需要 A → 依赖更深 → ...这是"自动化悖论"的直觉版本吗？(2) 这个循环是否可中断？什么条件下中断？(3) 哪些直觉子类型最容易陷入这个循环？哪些最抗拒？
> 
> 以下六个推演问题请逐一回答，每个附简要推理和置信度（高/中/低）。如涉及文献请标注。

## 六个推演问题

### Q1. A→B 正反馈循环的存在性

假说 A（LLM 作为直觉假肢）的实施是否会必然触发假说 B（人类直觉退化）？还是说 A→B 只在特定条件下发生？

具体而言：
- 航空自动化悖论中，飞行员手动飞行能力退化是因为**自动化替代了手动技能的执行**。LLM 作为"直觉假肢"替代的是什么？如果替代的是"直觉判断的执行"（人不再需要自己判断），退化应该发生。但如果替代的是"直觉判断的校验"（人仍然做判断，LLM 只检查对错），退化是否还会发生？
- **时间结构**是关键变量：GPS 是实时连续的自动化（每次导航都自动），结果导致空间直觉退化。LLM 的直觉假肢使用频率和模式是什么？是"每次都问"还是"只在不确定时问"？这两种使用模式对退化的影响是否不同？
- 是否存在**抗退化**的子类型？例如：社会型直觉必须通过实时互动维持——只要人还参与真实社会互动，LLM 假肢就无法削弱它？还是说即使参与真实互动，习惯了 LLM 的"标准答案"也会影响人在互动中的直觉敏感度？

### Q2. "双重崩溃"风险评估

假说 B 最危险的场景不是 LLM 替代了人类直觉（那至少还有 LLM 顶着），而是**LLM 削弱了人类直觉，但 LLM 自己在该领域也不擅长**——即"双重崩溃"（two-way degradation）。

推演问题：
- 四个直觉子类型中，哪些最容易出现双重崩溃？**提示**：双重崩溃需要两个条件：(a) LLM 在该子类型上有假肢级的可用性（让人愿意用），(b) LLM 在该子类型上有系统性盲区（不能完全替代人）。社会型直觉是否满足这两个条件？道德型呢？
- 航空自动化最危险的例子是"手动飞行技能退化 + 自动化在边缘情况失效"（如 AF447 空难——飞行员在自动驾驶断开后无法应对失速）。**LLM 直觉假肢的"边缘情况失效"会是什么样子？** 在社会型直觉上，什么场景下 LLM 的 sycophancy/omission bias 会导致灾难性失败，而此时人的直觉已经退化到无法接手？
- 过度自信传递（Liu et al. 2025）是 A→B 循环的具体机制之一。当 LLM 传递的过度自信让人误以为"LLM 的社会判断是可靠的"，而实际上 LLM 的社会判断在边缘场景中是偏差的——**这个 gap 有多大**？有没有量化估计？

### Q3. 中断循环的干预点

A→B 循环是否可以中断？如果可以，最有效的干预点在哪里？

推演问题：
- **设计干预点**：在 (a) "人是否咨询 LLM"、(b) "LLM 输出什么格式（给出答案 vs 给出选项 vs 只给校验）"、(c) "人做出自己的判断后再看 LLM 的答案"、(d) "使用频率"——哪一个最有效？
- 医学决策支持已有成熟的"第二意见"模式——医生先诊断，再看 AI 建议。这是否可以移植到直觉领域？"第二意见"模式是否足以防止退化？
- **退出策略**：一个人的直觉已经被 LLM 假肢削弱后，能否恢复？航空领域有"手动飞行复习训练"——如果停止使用 LLM 直觉假肢，直觉恢复需要多长时间？还是不可逆？

**附加说明**：
- 感知型直觉（如放射科读片）：LLM 假肢可以识别所有异常，人类只需确认 → 退化风险？这与自动驾驶飞行员退化是否同构？
- 社会型直觉：LLM 假肢提供文本社会知识（SJT 类）→ 退化风险？人在真实社交中还会自己判断吗？还是说人在真实互动中**被迫**使用直觉（不可回避性），因此退化风险低？

### Q5. 过度自信传递的直觉维度

Liu et al. (2025) + BIT Report (2025) 确认了 LLM 过度自信会传递给人类。但这个现象目前的证据主要在**知识型任务**上。

推演问题：
- 在**直觉型任务**上，过度自信传递会有什么不同？直觉判断的本质是快速、自动、伴随"确定感"——如果 LLM 带着过度自信给出直觉型判断（如"这个人是可信的"），人对 LLM 的信任结构是否不同？
- 直觉判断的一个特点是：人类通常附带**身体信号**作为校准（"gut feeling"）。如果人在收到 LLM 的社会判断后，自己的 gut feeling 与 LLM 不一致，人更可能**无视 LLM**（因为 gut feeling 不可回避）还是**压抑自己的直觉**（"AI 应该比我准"）？这取决于什么？
- 过度自信传递是否可能在**某些子类型上更强**？例如：在感知型上（"AI 视觉识别肯定比我好"——人类有自知之明，不太抵抗）vs 在社会型上（"AI 怎么比我更懂人际关系"——人类可能更抗拒）。

### Q6. 最激进的推演

如果一切极端化——LLM 直觉假肢在**所有四个子类型上**都以文本格式被广泛部署（即便社会型和道德型只有文本级可用性），使用频率极高（人类几乎每次社会判断前都咨询 LLM），且使用模式是"给出建议答案"而非"给出选项"或"校验已有判断"：

- **5 年后**，人类直觉在哪些子类型上已经不可逆地退化？哪些还存在残余？
- 此时如果发生**AI 系统大规模故障**（如模型后门攻击导致 LLM 社会判断系统性地输出有害建议），人类社会是否还能"退回"到自己的直觉？还是已经变成了直觉上的废人？
- 这个"最差情景"的可避免性取决于什么？当前有哪些社会机制（教育、职业培训、社交习惯）是天然防御？这些防御在多大程度上是自发的（不需要刻意保护）？
