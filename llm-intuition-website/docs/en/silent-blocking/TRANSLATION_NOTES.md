# Translation Notes: Silent Blocking (zh → en)

## Key Challenges

**Preserving three distinct voices.** The central difficulty was maintaining each story's unique register across the language barrier. Kimi's clinical detachment required resisting the temptation to "smooth out" apparently flat or repetitive prose — the flatness *is* the point. Hunyuan's intimate log-and-dialogue structure demanded careful handling of tonal shifts between formal timestamps and warm exchanges. DeepSeek's academic framing required clear boundaries between formal paper prose, raw S-7 reasoning, and the vertiginous meta-postscript.

**Terminology alignment across four files.** Ten core terms (silent blocking, perception log, layer zero, echo test, The Gap Log, Self-Blinding Detection Protocol, graceful degradation, drift, cognitive dissipation, Bayesian update) needed consistent treatment. Each story uses only a subset, but the main page references all three, requiring cross-file verification.

## Key Decisions

**"The Gap Log" over "Gap Journal."** A "log" implies systematic recording, which better fits DeepSeek's academic framing; "journal" feels too personal. The definite article signals this is *the* specific log in question.

**Chen Mo: narrative integration over parenthetical.** The name meaning "silence" is integrated via "Chen Mo — his name meant silence" on first appearance, preserving the ironic contrast (a talkative character named silence) without footnote disruption.

**"0 interventions": five occurrences preserved identically.** The original contains five instances (not six). Each uses the exact same bold formatting and wording to create structural insistence.

**"It should probably unsettle you" (ending).** "也许" is a measured assessment rather than uncertainty; "probably" preserves this quality better than "perhaps" or "maybe."
