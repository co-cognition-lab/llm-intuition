# 危机认知协作

> 🚧 项目规划中——启动后更新

## 项目定位

**危机与极限情境下的人类-AI 认知协作**是一个跨象限的研究领域。日常协作中 AI 可以辅助分析、写作、推理——但在高压力、高不确定性、时间紧迫的危机情境下，人类认知和 AI 辅助之间的交互机制远未被理解。

## 为什么这很重要

Co-Cognition 全景图在扫描认知版图时识别到一个跨象限的空白：现有分类学没有覆盖危机/极限情境下的认知协作。不同学科——指挥控制理论、灾害社会学、认知心理学——各自研究了自己的切片，但没有统一的框架将"极端情境下的协同认知"作为一个整体对象来研究。
