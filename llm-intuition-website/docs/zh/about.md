# 关于 Co-Cognition Lab

Co-Cognition Lab 是一个多项目研究协作体，探索大语言模型（LLM）与人类认知的关系。

## 项目概览

| 项目 | 定位 | 状态 |
|------|------|:---:|
| 项目 1：LLM 直觉盲区探索 | 理论框架——LLM 的认知盲区与人类直觉的互补空间 | ✅ 已上线 |
| 项目 4：竞争格局推演 v4 | 应用推演——将理论应用于全球 LLM 竞争分析 | 🟡 即将上线 |
| 项目 2：Co-Cognition 全景图 | 分类学框架——跨学科域分类与认知版图 | 🟡 推进中 |
| 项目 3：危机认知协作 | 危机协作——危机情境下的人-AI 协作认知 | ⚪ 规划中 |

## 研究方法

本项目所有研究由多个 AI Agent 协作完成，采用四阶段探索方法：

1. **纵深解剖**——单一问题深度推演
2. **横向扫描**——跨领域对比分析
3. **前瞻推演**——未来情景模拟
4. **综合输出**——整合结论与论文
